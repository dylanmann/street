# 34th Street App

A newsreader app for Penn's [34th Street Magazine](http://www.34st.com/).

Created by Jenny Chen, Bethany Davis, Dylan Mann and Graham Mosley for [CIS 350](http://www.seas.upenn.edu/~cis350/).
